# BBGV 修改列表

## Base 基础

### Improvement 改良

1. 渔船在工业化科技后 +1 生产力

### Resource 资源

1. 螃蟹修改为 +1 生产力 +1 金币，可能刷在礁石

## Civilization 文明

### Arabia 阿拉伯

1. 武阿拉伯使徒加力减少至 3，取消信仰购买建筑减价

### Korea 朝鲜

1. 取消书院基础相邻加成
2. 第一个总督头衔给予的书院建筑额外产出减少至25%；从第二个开始，总督头衔给予的书院建筑额外产出减少至15%

### Gran Colombia 大哥伦比亚

1. 大庄园提前至封建主义解锁，可修建在丘陵
2. 种植园 +1 生产力 +1 金币
3. 提高种植园关联至 T3

### Poland 波兰

1. 遗物的额外产出延后至政治哲学
2. 翼骑兵基础战斗力削弱至 59，解锁科技弹道学后 +5 战斗力

### Russia 俄罗斯

1. 所有市中心额外 +2 信仰

### Zulu 祖鲁

1. 三星延后至城市化
2. 伊坎达搓军团和军队不再减价
3. 作为补偿，伊坎达中的每级建筑额外 +1/+2/+4 生产力
